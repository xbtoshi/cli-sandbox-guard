# Security policy

Sandbox Guard is currently an alpha prototype. No version is declared production-ready.

Please use GitHub private vulnerability reporting for suspected vulnerabilities. Include:

- the Sandbox Guard commit or version;
- host OS, architecture, kernel or macOS version, Bubblewrap version, and Lima version when relevant;
- the smallest synthetic reproduction you can provide;
- the security invariant you expected and what occurred instead.

Do not attach real credentials or private repository contents.
